

# Ａは  Ｂです
注意这里的は是读 **wa**，是一种助词。です即“是”的意思。
は用于提示主题。

相当于 A是B
李さんわ，中国人です
わだしは，学生です

# ＡはＢ　ではありません
在口语中では也可以用じゃ代替
相当于 A不是B

森さんは学生　ではありません

# ＡはＢですか
相当于  A是B吗？
日语用か结尾表示疑问，没有疑问号

キムさんは中国人ですか。
回答：
はい，xxです
いいえ，ｘｘｘではありません
# ＡのＢ
助词の连接名词和名词。一般情况下是汉语的“的”的意思
李さんは大学の先生です

# 人称
わたし 我
あなた 你
あの人 那个人

あなた称呼对方比较不礼貌，仅在不知道对方姓名但必须招呼时采用。一对一聊天时可以不用。

さん不分男女，只用于他人的姓名之后，不用于自称。对于年龄比自己小或差不多的男性，可以用 君くん。

省略的例句(中文)：
(你是)吉田先生吗？
不是，(我)不是吉田。是森。


# 寒暄
